## Installation

### helm
```
 helm install triton.io https://github.com/triton-io/triton/releases/download/v0.1.1/triton-0.1.1.tgz
```

After a few seconds, if you get:

```
NAME: triton.io
LAST DEPLOYED: Sat Oct 23 22:28:58 2021
NAMESPACE: default
STATUS: deployed
REVISION: 1
TEST SUITE: None

```

Install triton success!
